# DropletUtils — Comprehensive Skill Guide

> **Domain:** Single-cell / droplet
> **Bioconductor:** [DropletUtils](https://bioconductor.org/packages/release/bioc/html/DropletUtils.html)
> **Paper:** Lun ATL et al. (2019). Genome Biology, 20:63.

Utilities for handling droplet-based single-cell RNA-seq data: emptyDrops for ambient RNA removal, barcodeRanks for knee detection, and read10xCounts for CellRanger output loading.

## When to Use

- Load 10x Genomics CellRanger output (barcodes/features/matrix)
- Distinguish real cells from empty droplets (emptyDrops)
- Detect knee/inflection points in barcode-rank plots
- Correct for ambient RNA contamination

**Alternatives:** `CellRanger filtered_feature_bc_matrix`, `SoupX (ambient RNA)`

## Do NOT Use When

- Plate-based scRNA-seq (Smart-seq2) — no droplet model needed.
- Pre-filtered CellRanger output — use raw (unfiltered) for emptyDrops.

## Data Requirements

| Requirement | Details |
|-------------|----------|
| **Input Format** | CellRanger raw_feature_bc_matrix/ directory OR raw count matrix |
| **Must Use Raw** | Use the UNFILTERED (raw) CellRanger output for emptyDrops; not the filtered output |
| **Min Cells Umi** | Barcodes with <100 UMI are used to estimate ambient profile |

## Workflows

### Load 10x Data and Remove Empty Droplets
*Read CellRanger output and use emptyDrops to filter real cells*

#### Load unfiltered matrix

```r
library(DropletUtils)
# Point to raw (unfiltered) CellRanger output
sce <- read10xCounts("path/to/raw_feature_bc_matrix/",
                      col.names=TRUE)
dim(sce)  # all barcodes, including empty
```

#### Inspect barcode-rank plot

```r
br_out <- barcodeRanks(counts(sce))
plot(br_out$rank, br_out$total + 1, log="xy",
     xlab="Rank", ylab="Total UMI", main="Barcode Rank Plot")
abline(h=metadata(br_out)$knee, col="blue", lty=2)
abline(h=metadata(br_out)$inflection, col="red", lty=2)
legend("bottomleft", c("Knee","Inflection"),
       col=c("blue","red"), lty=2)
```

#### Run emptyDrops

```r
set.seed(100)
e_out <- emptyDrops(counts(sce), lower=100, niters=10000,
                    BPPARAM=BiocParallel::SerialParam())

# Filter to real cells at FDR < 0.001
keep <- e_out$FDR <= 0.001
keep[is.na(keep)] <- FALSE
sce_filtered <- sce[, keep]
dim(sce_filtered)
cat("Cells retained:", ncol(sce_filtered), "
")
```

#### Downstream QC

```r
library(scater)
# Compute per-cell QC metrics
is_mito <- grepl("^MT-", rownames(sce_filtered))
df <- perCellQCMetrics(sce_filtered, subsets=list(Mito=is_mito))
colData(sce_filtered) <- cbind(colData(sce_filtered), df)

# Remove low-quality cells
qc_out <- quickPerCellQC(df, percent_subsets="subsets_Mito_percent")
sce_filtered <- sce_filtered[, !qc_out$discard]
cat("After QC:", ncol(sce_filtered), "cells
")
```

## Key Functions & Parameters

### `read10xCounts()()`

Load CellRanger output into a SingleCellExperiment

| Parameter | Description |
|-----------|-------------|
| `samples` | path(s) to the filtered_feature_bc_matrix/ directory |
| `col.names` | use barcode sequences as column names (default TRUE) |
| `type` | 'sparse' (default, memory-efficient) or 'HDF5' |

### `emptyDrops()()`

Test each barcode for ambient RNA; returns statistics for classification

| Parameter | Description |
|-----------|-------------|
| `m` | unfiltered count matrix (all barcodes) |
| `lower` | UMI threshold below which barcodes are assumed empty (default 100) |
| `niters` | Monte Carlo iterations for p-value calculation (default 10000) |
| `BPPARAM` | BiocParallel parameter for parallelization |

### `barcodeRanks()()`

Compute barcode-rank statistics for knee plot visualization

| Parameter | Description |
|-----------|-------------|
| `m` | count matrix |
| `lower` | lower bound for fitting |

## Scientific Assumptions

*The method assumes the following about your data and experiment:*

- Empty droplets contain only ambient RNA from the solution, not real cells.
- The Monte Carlo test in emptyDrops assumes the ambient profile can be estimated from low-UMI barcodes.
- Cells have significantly higher total UMI counts than empty droplets.

## Result Interpretation

- `emptyDrops()$FDR`: proportion of expected false discoveries; use FDR ≤ 0.001 for strict filtering.
- `barcodeRanks()$knee`: UMI count at the knee of the rank curve — approximate threshold for cell/empty distinction.
- `barcodeRanks()$inflection`: lower bound for cell-containing droplets.
- Cells called by emptyDrops at FDR<0.001 can include cells with very low UMI that CellRanger missed.

## Best Practices

- Use `emptyDrops()` on the UNFILTERED CellRanger output (raw_feature_bc_matrix/) for best sensitivity.
- Filter `emptyDrops()` results: `keep <- e.out$FDR <= 0.001 & !is.na(e.out$FDR)`.
- Check the knee plot (`barcodeRanks`) before choosing the `lower` UMI threshold for emptyDrops.
- After emptyDrops, run `sce <- sce[, keep]` to subset to real cells.
- For standard 10x data, `emptyDrops(lower=100, niters=10000)` is a good default.

## Common Errors & Troubleshooting

### `Error in emptyDrops: all barcodes have total counts <= lower`

**Cause:** `lower` parameter set too high relative to data

**Fix:** Reduce `lower`; inspect `barcodeRanks(m)` to set appropriate value

### 1 Introduction


Droplet-based single-cell RNA sequencing (scRNA-seq) technologies allow researchers to obtain transcriptome-wide expression profiles for thousands of cells at once.
Briefly, each cell is encapsulated in a droplet in a oil-water emulsion, along with a bead containing reverse transcription primers with a unique barcode sequence.

Bursting of the droplets yields a pool of cDNA for library preparation and sequencing.
Debarcoding of the sequences can then be performed to obtain the expression profile for each cell.
This package implements some general utilities for handling these data after quantification of expression.

### 4 Computing barcode ranks


A useful diagnostic for droplet-based data is the barcode rank plot, which shows the (log-)total UMI count for each barcode on the y-axis and the (log-)rank on the x-axis.
This is effectively a transposed empirical cumulative density plot with log-transformed axes.
It is useful as it allows users to examine the distribution of total counts across barcodes, focusing on those with the largest counts.
To demonstrate, let us mock up a count matrix:
We compute the statistics using the barcodeRanks function, and then create the plot as shown below.
The knee and inflection points on the curve mark the transition between two components of the total count distribution.
This is assumed to represent the difference between empty droplets with little RNA and cell-containing droplets with much more RNA

```r
set.seed(0)
my.counts <- DropletUtils:::simCounts()
```

```r
br.out <- barcodeRanks(my.counts)

# Making a plot.
plot(br.out$rank, br.out$total, log="xy", xlab="Rank", ylab="Total")
o <- order(br.out$rank)
lines(br.out$rank[o], br.out$fitted[o], col="red")

abline(h=metadata(br.out)$knee, col="dodgerblue", lty=2)
abline(h=metadata(br.out)$inflection, col="forestgreen", lty=2)
legend("bottomleft", lty=2, col=c("dodgerblue", "forestgreen"), 
    legend=c("knee", "inflection"))
```

### 5 Detecting empty droplets


Empty droplets often contain RNA from the ambient solution, resulting in non-zero counts after debarcoding.
The emptyDrops function is designed to distinguish between empty droplets and cells.
It does so by testing each barcodeâs expression profile for significant deviation from the ambient profile.

Droplets with significant deviations from the ambient profile are detected at a specified FDR threshold, e.g., with FDR below 1%.
These can be considered to be cell-containing droplets, with a frequency of false positives (i.e., empty droplets) at the specified FDR.
Furthermore, droplets with very large counts are automatically retained by setting their p -values to zero.
